# 🚗 Vehicle Rental System (Java OOP Project)

## 📌 Concepts Covered
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Method Overloading & Overriding
- Dynamic Method Dispatch
- Upcasting & Downcasting

## 🏗️ Project Structure
- model → Core classes
- service → Business logic
- utils → Helper classes

## ▶️ How to Run
1. Compile:
   javac com/rental/Main.java
2. Run:
   java com.rental.Main

## 💡 Purpose
This project demonstrates core Java OOP concepts in a simple rental system.
